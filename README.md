# Shelly's Bistro Shift Management App

A simple and efficient shift scheduling and staff management application designed for Shelly’s Indigenous Bistro.

## 🚀 Overview

This app helps manage daily restaurant operations by organizing:
- Staff schedules
- Shift assignments
- Time-off and swap requests
- Team announcements
- Weekly hours tracking

Built with React for fast, responsive performance.

---

## 🧩 Features

- 📅 Weekly shift scheduling board
- 👥 Staff directory with roles
- 🔄 Shift request management (approve/decline)
- 📢 Team announcements feed
- ⏱ Weekly hours tracking per staff
- ➕ Easy shift and staff creation

---

## 🏢 Use Case

Designed specifically for restaurant operations including:
- Front of House
- Kitchen Line
- Pizza Room
- Catering Team
- Management Oversight

---

## ⚙️ Tech Stack

- React
- Tailwind CSS
- ShadCN UI Components
- Lucide Icons
